# Angular Chat App

## Dependencies
* `npm install express`
* `npm install angular`
* `npm install socket.io`

## Angular Special Components
* `ng generate service socket`

## Socket.IO Bug
* In `polyfill.ts`, add:
```javascript
(window as any).global = window;
```
